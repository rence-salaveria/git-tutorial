# Git Tutorial

This is a tutorial on how to use git and github.com

## Git Workflow

- Create a repository (git init) FIRST TIME ONLY
- git add
- git commit -m

## GitHub Workflow (Personal)

- git init (FIRST TIME ONLY)
- git add
- git commit -m
- git push

## Github Workflow (Collaboration)

- git pull
- git add
- git commit -m
- git push
